# Requirements

This project does not require any external libraries beyond Python/C++ standard libraries.

## C++ Version
To run the C++ version:
1. A C++ compiler (like g++ from MinGW or MSVC)
2. A Windows Operating System (since it uses <windows.h> for threading)

Compilation Command:
g++ parallel_quicksort.cpp -o parallel_quicksort.exe

## Python Version
To run the Python version:
1. Python 3.6 or higher (uses standard library only)
2. Cross-platform (works on Windows, Linux, macOS)

Run Command:
python parallel_quicksort.py
